Informacje o identyfikatorze ID w CSS

Pojęcie Znaczenie pojęcie uzupełnia uczeń
Co znaczy skrót ID? ID - Identyfikator używany do nadania unikalnej nazwy elementowi HTML.
Jaki jest cel stosowania ID? Identyfikator służy do wyróżnienia konkretnego elementu na stronie i przypisania mu indywidualnego stylu.
Jakie jest znaczenie znaku # W CSS przed nazwą identyfikatora stosujemy znak #, aby go zdefiniować lub wywołać (np. #nazwaID).
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W HTML: id="nazwa"
W CSS: #nazwa { ... }
Wywołanie polega na przypisaniu atrybutu id do elementu oraz odniesieniu się do tego ID w arkuszu stylów CSS za pomocą #nazwa.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  • Element z danym ID powinien wystąpić tylko raz w dokumencie HTML.
  • Jeśli dany styl ma być stosowany wielokrotnie, lepiej użyć class zamiast id.
  • Powtarzanie tego samego ID jest błędem strukturalnym, ale nie zawsze powoduje błędne działanie strony.
