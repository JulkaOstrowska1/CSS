Ostrowska - CSS pozwala oddzielić treść strony od sposobu jej prezentacji.
Dzięki stylom można zarządzać wyglądem całej strony z jednego pliku.
To pozwala na łatwiejsze utrzymanie i edycję kodu.
Style mogą być wewnętrzne, zewnętrzne lub wbudowane.

Ostrowska - HTML to język znaczników służący do tworzenia stron WWW.
Składa się z tagów, które opisują strukturę dokumentu.
Każdy element HTML ma swój określony cel i znaczenie.
Przeglądarki interpretują te znaczniki i wyświetlają stronę.

Ostrowska - Internet to globalna sieć komputerowa umożliwiająca wymianę informacji.
Strony WWW są podstawowym źródłem danych w Internecie.
Technologie takie jak HTML, CSS i JS tworzą dynamiczne serwisy.
Znajomość tych języków jest kluczowa dla twórców stron.

Ostrowska - JavaScript pozwala dodawać interaktywność do stron.
Można nim reagować na akcje użytkownika i modyfikować DOM.
JS współpracuje z HTML i CSS tworząc pełne aplikacje webowe.
Dzięki bibliotekom i frameworkom jak React rozwój jest łatwiejszy.
